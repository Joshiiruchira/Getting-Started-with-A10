The A10 Control interface is clean and easy to use. It helps you quickly find options, manage devices, and monitor system performance.

## Screen Overview
1. Top Bar

On the top bar, you can see the following:

- View notifications and alerts
- Access help and documentation
- Open your user profile and settings
- Log out

It also shows your selected organization and user role.

2. Left Pane

On the left pane, you can view:

- Dashboard – Shows system health and performance summary
- Devices – Add and manage devices
- Applications – Manage application services
- Analytics – View traffic and performance reports
- Alerts – Check system warnings and errors
- Administration – Manage users, roles, licenses, and system settings

3. Dashboard

The dashboard helps you monitor live system information using charts and numbers. You can view the following options:

- Device health
- Application traffic
- System performance
- Security events

4. Main Screen

You can view the following options:

- Configure settings
- View reports
- Manage devices and applications

5. Action Buttons

You can perform common tasks using these options:

- Add
- Edit
- Delete
- Refresh
- Download

6. Search and Filters

Search and filter options help you find devices, users, and reports, especially when you have multiple entries.
