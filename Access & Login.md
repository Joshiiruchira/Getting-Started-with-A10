This section explains how to log in to A10 Control for the first time using both the web interface (GUI) and the command-line interface (CLI).

1. First-Time Login

After A10 Control installation is complete, the system is ready for login.

You can access A10 Control using:

1. GUI is recommended for all configuration and daily tasks

2. CLI is used mainly for troubleshooting and maintenance

Log in using GUI

Follow these steps to log in using a web browser:

1. On the Login page, enter the default organization name:
2. Click Next.
3. On the Organization Login Credentials page, enter your username and password.
4. Click Login.
5. When prompted, reset the password.
6. Review and accept the License Agreement to continue.
7. After successful login, the A10 Control Home page is displayed.

Log in using CLI 

The CLI console is used for troubleshooting and maintenance tasks.

To log in using the CLI console, enter your username and password.
To change the CLI password, run:

sudo passwd

Note: The CLI password and GUI password are different. Changing one does not change the other.
